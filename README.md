# garnetcss

An enterprise-ready, modern, unobtrusive and elegant semantic-css framework (<100KB) based on emeraldcss.

# Quick Start

## HTML Quick Start

```html
<!doctype html>
<html>
<head>
    <title>CSS</title>
    <link rel="stylesheet" href="/main.css" />
</head>
<body>
    <header>
        <h1>CSS</h1>
    </header>
    <main></main>
    <script type="module" src="/main.js" defer></script>
</body>
</html>
```
